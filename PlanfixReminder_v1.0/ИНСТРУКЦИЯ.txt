🚀 НАСТРОЙКА ЗА 3 ШАГА:

┌─ 1. ПОЛУЧИТЕ API ТОКЕН ────────────────────────────────────────┐
│ • Зайдите в ваш Planfix: https://your-account.planfix.com     │
│ • Настройки → API → REST API                                   │
│ • Создайте токен с правами "Чтение задач"                      │
│ • Скопируйте токен (показывается только один раз!)             │
└────────────────────────────────────────────────────────────────┘

┌─ 2. НАСТРОЙТЕ ПРОГРАММУ ───────────────────────────────────────┐
│ • Скопируйте "config.ini.example" → "config.ini"              │
│ • Откройте config.ini в блокноте                               │
│ • Замените "ВСТАВЬТЕ_ВАШ_API_ТОКЕН_ЗДЕСЬ" на ваш токен        │
│ • Замените "l-s" на название вашего аккаунта                   │
│ • Сохраните                                                    │
└────────────────────────────────────────────────────────────────┘

┌─ 3. ЗАПУСТИТЕ ─────────────────────────────────────────────────┐
│ • Двойной клик по PlanfixReminder.exe                         │
│ • Программа запустится в фоне                                  │
│ • Уведомления появятся в правом верхнем углу                   │
└────────────────────────────────────────────────────────────────┘

🎨 ЦВЕТА УВЕДОМЛЕНИЙ:
🔴 КРАСНЫЕ (просроченные) - висят пока не закроете + звук
🟡 ОРАНЖЕВЫЕ (срочные) - висят пока не закроете + звук  
📋 СИНИЕ (обычные) - висят пока не закроете, без звука

🎛️ КНОПКИ В ОКНАХ:
• Перетаскивание - хватайте за заголовок
• 📌 - закрепить/открепить поверх окон
• ✕ - закрыть (появится снова через время)
• "Открыть" - открыть задачу в Planfix
• "15мин" / "1ч" - отложить уведомление
• "Готово" - пометить как просмотренную

❗ ПРОБЛЕМЫ:
• Нет config.ini → создайте из config.ini.example
• Нет токена → получите в настройках Planfix
• Не подключается → проверьте токен и URL
• Нет задач → убедитесь что вы исполнитель/постановщик

🛑 ОСТАНОВКА: Найдите программу в системном трее (у часов)
